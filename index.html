Firewall Rule Example
access-list OUTSIDE-IN extended permit tcp any any eq 80

# Add a firewall rule to allow HTTP traffic

Advanced: Object-Based NAT Configuration
object network obj_any
  nat (inside,outside) dynamic interface
            
# Configure dynamic NAT using an object
object network obj_internal
  subnet 192.168.1.0 255.255.255.0
